Project Definition & Objectives

Project Definition

The project aims to build a multi-tenant Attendance SaaS platform that integrates with Technospy's biometric devices (eSSL, ZKTeco, BioMax) while also supporting standalone mobile/web attendance (QR code, geofencing, selfie + liveness).

Primary Objectives

  • Seamless Device Integration with eSSL, ZKTeco, BioMax via push/ADMS or site gateway
  • Multi-Tenant Cloud Platform supporting multiple companies with their own policies
  • Modern Employee Experience with mobile/web apps for punching, leave, notifications
  • Powerful Admin Dashboard for HR, managers, and payroll teams
  • Recurring Revenue Model so you can bundle software with your hardware offering

Scope Summary

Web Admin Panel

  • Organization setup, users, locations
  • Device onboarding & health monitoring
  • Shift & policy setup, holiday calendars
  • Approvals (regularization, leave, OT)
  • Reports, dashboards, and payroll exports

Mobile App

  • Punch-in/out (geo/QR/selfie), leave requests
  • Notifications and attendance calendar view

Core Features

  • Multi-tenancy with subdomain-based access
  • Role-based access control
  • REST/GraphQL APIs & Webhooks
  • DPDP-compliant data handling
