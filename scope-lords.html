Project Overview

Objective

The Lord's Path Digital Platform is a comprehensive healthcare solution designed to streamline the patient journey — from diagnostic test booking and sample collection to smart health reports, doctor consultations, and digital prescriptions. The platform will consist of four integrated applications that work seamlessly together:

User Mobile Application

To empower patients to seamlessly manage their complete health journey while enabling Lord’s Path to strengthen its brand presence through strategic marketing initiatives.

Lab Admin Web Application

For labs to manage operations, bookings, and resources

Phlebotomist Mobile Application

For sample collection agents to manage assigned requests

Doctor Application

For doctors to manage consultations and prescriptions

Scope of Work

1. User Mobile Application (iOS & Android)

Authentication & Profile

  • OTP-based login
  • Create and edit user profile

Health Insights & Recommendations

  • Organ health scores based on user inputs
  • Test and package recommendations driven by health data

Lab Tests & Reports

  • Book lab tests (with or without smart reports)
  • End-to-end test status tracking (temperature showcase for certain tests)
  • View and download reports and smart reports
  • Purchase test packages

Teleconsultation & Care

  • View doctor availability
  • Chat and video consultation
  • Digital prescription generation
  • Language-based doctor matching

Payments & Engagement

  • Payment gateway integration
  • Push notifications for status updates, reminders, and new recommendations

2. Lab Admin Web Application

  • Role-based login (Admin, Super Admin)
  • Create, view, and edit lab details
  • Manage lab tests and health packages (including One DNA tests)
  • Add, view, or remove doctors and phlebotomists
  • Auto-assign test requests to phlebotomists based on location and availability
  • View and manage all bookings
  • Report generation and management

3. Phlebotomist Mobile Application

  • OTP login and profile management
  • View pending and completed requests
  • Mark availability
  • Update request status (accepted, rejected, on the way, sample collected, sample deposited)
  • Push notifications for new requests and updates

4. Doctor Application

  • OTP login and profile management (including language)
  • View past consultations
  • Mark availability and accept new requests
  • Conduct video and chat consultations
  • Create and share prescriptions
